Klases

Viena klase gali buti skirta daugeliui elementu(tag'u). Vienas elementas gali turetu daug klasiu. Klasese nurodytas vienodas stilius yra prioritizuojamas pagal tai, kelintas eileje jis yra aprasytas faile. Pavyzdiui jeigu tam paciam elementui su dviem klasem bus duotos spalvos, tai vyraus spalva kuri buvo paskutine nurodyta CSS.
